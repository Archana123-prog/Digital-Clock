# ⏳ Smart Digital Clock & Timer App

A modern and interactive **Digital Clock + Alarm + Timer Web App** built using **React.js** with advanced features like animations, persistent storage, and real-time updates.

## 🚀 Live Features

https://digital-clock-murex-three-11.vercel.ap

### 🕐 Clock

* Real-time clock (updates every second)
* 12-hour / 24-hour format toggle
* Displays current date & day

### ⏰ Alarm

* Set multiple alarms
* Alarm sound + voice alert 🔊
* Notification popup 🔔
* Data saved in localStorage

### ⏳ Timer

* Countdown timer (minutes + seconds)
* Smooth UI with animations
* Works in background (tab switching)
* Alarm triggers when time is up
* Persistent timer using localStorage

### 🌍 Timezone

* Shows time for multiple countries
* Auto-detect user timezone

### 🎨 UI & UX

* Glassmorphism premium design
* Dark theme 🌙
* Fully responsive 📱
* Animated transitions using Framer Motion
* Lottie animation (sand clock ⏳)

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS3 (Glass UI)
* 🎬 Framer Motion (animations)
* 🎥 Lottie React (animations)
* 💾 LocalStorage (data persistence)
* 🔔 Web APIs (Notification + Speech)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/digital-clock-app.git
```

2. Navigate to project

```bash
cd digital-clock-app
```

3. Install dependencies

```bash
npm install
```

4. Run development server

```bash
npm run dev
```

## 🔔 Important Permissions

* Enable **Notifications** for alarm alerts
* Click once to enable **Sound playback** (browser restriction)
